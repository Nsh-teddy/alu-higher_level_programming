# Python - Input/Output

## Task 0: Read file

This task implements a function `read_file(filename="")` that reads a UTF-8 encoded text file and prints its contents to standard output.

### Requirements
- Use the `with` statement for file handling.
- Do not handle file permission or non-existent file exceptions.
- No external modules allowed.
- Compatible with Python3.

### Files
- `0-read_file.py`: Contains the `read_file` function implementation.
- `0-main.py`: Script to manually test the function.
- `my_file_0.txt`: Sample file for testing output.
