# Python - More Data Structures: Set, Dictionary

This project focuses on working with more advanced data structures in Python, specifically sets and dictionaries.

## Project Structure

All Python files are written in Python 3 and are executable from the command line.

Each file is named by its task number and contains a function that fulfills a specific requirement.

## Tasks Covered

- Square values in a 2D matrix
- Accessing elements safely in lists
- Modifying and copying lists
- Working with dictionaries (creation, update, deletion)
- Using sets for operations like union, intersection, and difference
- Lambda, map, filter, and reduce functions

## Requirements

- Python 3
- Files start with the shebang `#!/usr/bin/python3`
- No external libraries or modules are imported
- Functions follow the provided prototypes
