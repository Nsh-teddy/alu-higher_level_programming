# Python - More Classes and Objects

This project focuses on deepening understanding of object-oriented programming (OOP) concepts in Python. It builds on the basics of classes and objects by introducing more advanced features and practices.

## Topics Covered

- Class and instance attributes
- Special methods: `__init__`, `__str__`, `__repr__`, `__del__`
- Encapsulation and data hiding
- Property decorators (`@property` and `@setter`)
- Class methods and static methods
- Comparing objects using special comparison methods
- Custom string representations and clean-up actions
