# Python Classes - Task 0: My First Square

## Overview

This project contains Python tasks focused on classes. The first task is to create an empty class named `Square` which defines a square.

## Task Description

Write an empty class called `Square`. This class should not contain any attributes or methods. The purpose is to define the class structure only.

- No imports are allowed.
- Use `pass` to create an empty class.
- Follow Python naming conventions and style guidelines.

## Files

- `0-square.py`: Contains the definition of the `Square` class.
- `0-main.py`: Test file that imports and creates an instance of the `Square` class, then prints the type and dictionary of the instance.

## How to Run

Make sure both files are in the same directory. Then execute:

```bash
chmod +x 0-main.py
./0-main.py
